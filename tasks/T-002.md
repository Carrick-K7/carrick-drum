## 📋 需求背景

Drum 核心功能已完成，现开发 **MP3 导入功能**：允许用户上传 MP3 文件，系统自动识别鼓点节奏并生成可练习的鼓点谱。

---

## 🎯 功能目标

### MVP 版本
1. **MP3 上传**
   - 支持拖拽上传或文件选择
   - 文件大小限制：最大 20MB
   - 格式限制：MP3  only

2. **音频分析**
   - 使用 Web Audio API 提取音频特征
   - 识别强拍/重音作为鼓点
   - 生成时间戳序列

3. **鼓点谱生成**
   - 将识别结果转为 Drum 可播放的谱面格式
   - 简化版：自动映射到基础鼓组（kick/snare/hihat）

4. **预览与编辑**
   - 播放原始 MP3 同步显示识别的鼓点
   - 支持微调鼓点位置
   - 保存为自定义课程

---

## 🔧 技术方案

### 核心库选择
- **Web Audio API** - 音频解码和分析
- **Peak.js** 或自研 - 音频可视化
- ** onset detection 算法** - 鼓点识别（基于能量突变）

### 数据结构
```typescript
interface MP3ImportResult {
  id: string
  fileName: string
  duration: number
  bpm: number
  drumPoints: DrumPoint[]
  createdAt: number
}

interface DrumPoint {
  time: number      // 时间点（秒）
  strength: number  // 强度 0-1
  autoDrum: string  // 自动分配的鼓：kick/snare/hihat
  confirmed: boolean // 用户是否确认
}
```

### 处理流程
1. 用户上传 MP3
2. Web Audio API 解码音频
3. 低通滤波提取低频（kick）
4. 高通滤波提取高频（hihat）
5. 中频分析提取 snare
6. 生成时间戳序列
7. 用户预览/编辑
8. 保存为自定义课程
