## 🔧 技术要点

1. **节奏数据格式**: 统一使用JSON格式存储节奏映射
2. **BPM递增**: 从60逐步提升到120，平滑过渡
3. **难度曲线**: 合理分布，避免跳跃过大
4. **复用性**: 节奏数据可用于伴奏模式

### 节奏数据结构示例
```typescript
interface RhythmData {
  id: string;
  name: string;
  bpm: number;
  timeSignature: [number, number];
  measures: number;
  notes: Array<{
    time: number;        // 相对开始时间的毫秒
    drum: DrumType;      // 鼓件类型
    velocity?: number;   // 力度 (0-1)
  }>;
  metadata: {
    difficulty: 'beginner' | 'elementary' | 'intermediate' | 'advanced';
    targetDrums: DrumType[];
    description: string;
    tips: string[];
  };
}
```
